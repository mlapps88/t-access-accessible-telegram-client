# t-access-accessible-telegram-client

**TAccess** is a custom Telegram client built with Python and wxPython, specifically designed to be fully accessible for screen reader users (NVDA, JAWS, Narrator) on Windows.

## üåü Key Features

*   **Accessibility First**: Every element is navigable via keyboard. Custom focus management and screen reader announcements ensure a smooth experience.
*   **Efficient Navigation**:
    *   `Arrow Keys`: Navigate chat lists and messages.
    *   `Tab`: Move between filters, search, and input fields.
    *   `F1`: Open the built-in User Guide.
*   **Productivity Shortcuts**:
    *   `Ctrl+S`: Global Search (Find users, groups, channels).
    *   `Ctrl+Shift+A`: Attach files quickly.
    *   `M`, `A`, `P`: Mute, Archive, or Pin chats directly from the list.
    *   Interaction with Bots and inline buttons.
*   **Portable**: No installation required. Runs directly from the folder.
*   **Multi-language**: Fully localized in English, Italian, and Spanish.

## üì• Installation

TAccess is valid as a **Portable Application**.
1.  Go to the [Releases](../../releases) page.
2.  Download the latest `TAccess_Portable.zip`.
3.  Extract the convenient folder to a location of your choice.
4.  Run `TAccess.exe`.

## üõ† Usage

*   **First Run**: You will be asked to log in using your phone number and the code sent to your Telegram account.
*   **Updating**: The application includes an auto-updater. It will check for new versions on startup and prompt you to update.
